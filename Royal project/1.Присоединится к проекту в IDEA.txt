При открытии IDEA в первый раз выбрать Project from VCS
Если ранее работали в IDEA выбрать File - New - Project from Version Control

ИДЕЯ предложит авторизоваться на Github. У вас должен быть аккаунт на гитхабе и доступ к проекту.

Не забывайте что вам понадобится JDK для работы с кодом java, если JDK не установлен вы увидите ошибки в проекте из-за невозможности обращения к классам java.lang , java.util
Установить JDK можно в свойствах проекта. Для этого проекта использовался JDK11

Запустите проект чтобы убедится в его работоспособности. 
Проект ориентирован на сборку при помощи Maven и может не запуститься сборщиком Ant, который используется в IDEA по-умолчанию. Чтобы поменять сборщик по-умолчанию для проекта используем настройку 
File - Settings - Build... - Build Tools - Maven - Runner 
Проставляем флаг Delegate EDE build/run actions to Maven

Во время старта проекта стартует встроенный в проект вебсервер tomcat , он покажет номер порта на котором будет развернуто веб приложение. Сейчас это порт 8888

Проверяем работоспособность приложения вводим в бразер строку

http://localhost:8888/greeting?name=Novice

В ответе видем JSON текст

{"id":1,"content":"Hello, Novice!"}

Где id это номер обращения (по порядку) , а надпись Hello, Novice собрана из шаблона с параметром указанным в запросе.
